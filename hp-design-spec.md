参照: スマートスマイル薬局（シングルページ・LP型）

---

## 1. サイトの型

| 項目 | 内容 |
|------|------|
| ページ構成 | **1ページ完結**（縦スクロール） |
| 目的 | 来局前の不安解消・LINE処方せん・問い合わせ導線 |
| トーン | 清潔・親しみ・信頼（イラスト＋余白多め） |
| 想定幅 | デスクトップ中心（最大幅 960〜1100px 前後） |

| 役割 | 用途 | 目安 |

| プライマリ青 | ヘッダー・フッター・見出し・枠線 | `#0099D8` 系 |
| アクセント緑 | LINE CTA・QR周辺 | `#00B900` / `#06C755` 系 |

## 3. タイポグラフィ

- **フォント**: ゴシック系（Noto Sans JP / ヒラギノ / system-ui）
- **H1（ヒーロー）**: 大・青・2〜3行改行可
- **セクション見出し**: 青・中央寄せ・太字
- **カード見出し**: 青・太字・1行〜2行
- **本文**: 14〜16px・行間 1.6〜1.8
- **電話番号（アクセス）**: 大きめ・青・クリックで `tel:`

## 4. セクション構成（上から順）

### 4.1 ヘッダー（固定推奨）

```
[ロゴ + 薬局名]     [電話番号]  [LINEで処方せん送信（緑ボタン）]
                    [よくあるご質問 | お支払い方法 | アクセス]（アンカー）
```

- 背景: プライマリ青・文字白
- **必須CTA**: LINE処方せん（ヘッダー右上で常時見える）
- ナビ: ページ内アンカー（`#faq` `#payment` `#access` 等）

### 4.2 ヒーロー

- 背景: 薄イエロー
- **2カラム**: 左テキスト / 右イラスト（薬剤師）
- 含める要素:
  - キャッチコピー（健やかな暮らしのパートナー等）
  - 営業時間の要約（平日・土日で分ける）
  - 地域・安心感の一文

### 4.3 特徴グリッド「〇〇薬局なら」

- 背景: 白
- **カード 7〜8個**（4列×2行 → 下段は3個中央寄せでも可）
- カード: 白背景・水色細枠・角丸・パディング統一
- 典型トピック:
  1. どの病院の処方せんでも受付
  2. マイナンバーカード（資格確認）
  3. 在宅・訪問
  4. お薬相談
  5. 一包化
  6. LINE受付
  7. お支払い方法

### 4.4 LINE誘導

- 背景: 薄イエロー
- 見出し: **緑**（LINEブランドに合わせる）
- 中央: **QRコード（大）**
- 左右: スマホ利用イラスト（装飾）
- サブコピー: 友だち追加・相談の流れ

### 4.5 FAQ「お薬についてよくあるご質問」

- 形式: `Q:`（紫）+ 質問 / `A:`（青）+ 回答
- 3問程度から開始可
- **表**: 食前・食後・食間などの用語表（1問目横または下）
- テーブル: 青ヘッダー・青枠線

### 4.6 お支払い方法

- 背景: 薄ブルー
- テキスト: 現金・クレジット・電子マネー（iD / QUICPay 等）
- **カードブランドロゴ横並び**（VISA / MC / AMEX / JCB / Diners）

### 4.7 アクセス

- **2カラム**: 左 Googleマップ埋め込み / 右 店舗情報
- 右カラム必須項目:
  - 薬局名
  - 営業時間（平日・土日祝で表記分け）
  - 住所・郵便番号
  - 最寄駅からの徒歩
  - 電話（大きく）

### 4.8 お知らせボックス

- 青ヘッダーバー + 白/グレー本文エリア
- 例: マスク着用方針・感染症対応・臨時休業

### 4.9 フッター

- 背景: ヘッダーと同じ青
- 中央: `© 年号 薬局名`（白・小さめ）

## 6. レイアウト・レスポンシブ

| ブレイクポイント | 挙動 |
|------------------|------|
| PC | 2カラム（ヒーロー・アクセス）・特徴4列 |
| タブレット | 特徴2〜3列 |
| スマホ | 1カラム縦積み・ヘッダーは電話+LINEを縦またはハンバーガー検討 |

- **Grid**: 特徴セクション → `grid-template-columns: repeat(4, 1fr)` → モバイルで `1fr`
- **Flex**: ヘッダー上下2段、カード内中央揃え

## 7. コンテンツ・SEO・運用

- **title / description**: 地域名 + 薬局名 + 営業時間
- **構造化データ**: `Pharmacy` + `OpeningHoursSpecification`
- **OGP**: 店舗外観 or イラスト
- **必須リンク**: LINE公式、Googleマップ、電話
- **更新しやすい領域**: お知らせボックス・営業時間・FAQ
